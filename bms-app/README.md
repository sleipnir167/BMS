# 業務管理システム（BMS）モックアプリ

DX推進プロジェクト向け業務管理Webアプリケーションのモックです。  
要件定義書（REQ-BMS-2025-001）・機能設計書（FDS-BMS-2025-001）に基づいて実装しています。

---

## 📦 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | React 18 |
| ビルドツール | Vite 5 |
| チャートライブラリ | Recharts |
| スタイリング | CSS-in-JS（グローバルCSS変数） |

---

## 🚀 セットアップ

### 必要環境
- Node.js 18 以上
- npm 9 以上

### インストール・起動

```bash
# リポジトリをクローン
git clone <repository-url>
cd bms-app

# 依存パッケージのインストール
npm install

# 開発サーバー起動（ http://localhost:3000 ）
npm run dev

# プロダクションビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## 🎯 実装機能一覧

### ダッシュボード
- KPI 4指標（申請件数・承認率・承認待ち・経費合計）
- 月次申請トレンド折れ線グラフ
- 申請種別割合ドーナツグラフ
- 最近の申請一覧
- アクティビティログ

### 申請管理
- 申請一覧（ステータス・種別フィルタ、キーワード検索）
- 新規申請作成（タブ形式フォーム・バリデーション）
- 下書き保存 / 申請提出
- 申請詳細モーダル（承認フロー可視化・コメント）
- 申請取下げ / 削除

### ワークフロー
- 承認待ち一覧
- 承認 / 差戻し / 却下 アクション
- コメント必須バリデーション（差戻し・却下時）
- 承認フローステップ表示

### ユーザー管理
- ユーザー一覧（ロール別バッジ）
- ユーザー追加・編集・削除

### マスタ管理
- 部門マスタ CRUD
- 勘定科目マスタ CRUD・有効/無効切替

### レポート
- 部門別経費実績 vs 予算グラフ
- ステータス別集計プログレスバー
- 申請一覧テーブル
- CSV / PDF エクスポートボタン（UI のみ）

### 共通
- 通知パネル（既読管理）
- トースト通知（成功・エラー）
- ダークテーマ UI

## ⚠️ 注意事項

本アプリはモック（デモ）です。以下の機能は UI のみ実装しています：
- ファイルアップロード
- CSV / PDF エクスポート
- メール通知
- 外部システム連携（人事・会計システム等）

本番実装時は `/src/data/mockData.js` の各データを実際の API 呼び出しに置き換えてください。
